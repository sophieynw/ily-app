# Valentine's Day 2026

A tiny Valentine’s Day web app built with React + TypeScript + Vite. Try one of the following names: Lucy, Wing, Oscar, Tubby, Alex, Monica.

## What it does
- Home page where you enter a name
- Fetches a love letter from an API
- Displays the letter on the “Ily” page

## Tech
- React + TypeScript
- Vite
- React Router
- TanStack React Query

## Run locally
```bash
npm install
npm run start-dev
```
## Build
```bash
npm run build
npm run preview
```
